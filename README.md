# iTunesQueryDisplay
An ITunes Query Search Display

PLEASE DO NOT COMMIT TO THIS PROJECT

3 Parts

1. Globe
2. Pick a Country
3. Display IT ALL :)


Objects:

1. Globe Object
  - Map of the World with a scrollable option
  - Passes the World String to the Search Class
2. Search
  - Recieves a String 
  - Searchs the ITunes API 
  - Returns a JSON List
3. Display
  - Displays the JSON List 
  - Shows the Songs
